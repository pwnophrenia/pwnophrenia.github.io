Welcome

Welcome, fellow hackers! I hope you’re all doing well because this course is designed to challenge you a bit
Honestly, I aimed to make it as straightforward as possible, but let’s face it, low-level stuff can be tough, so get ready for that! Before we dive in, please make sure you have all the prerequisites.

This course is dedicated ONLY to vulnerabilities, nothing else.

Prerequests

Before starting, I hope you know:

  • Basic C Programing
  • Basic Assembly Language (Any one)
  • x86-64 CPU
  • Willness, patience, and guts

Boring Rules

There is also some boring, but IMPORTANT rules.I hope you will follow them, if you really want to learn something and challenge yourself.

  1. Don't use LLM (AI)
  2. Don't rush
  3. Don't be shy to ask help from HUMANS
  4. GOOGLE THINGS THAT YOU DON'T UNDERSTAND

Let's Go

This course is set up for self-learning right from the start. I’ll provide you with some initial materials to help you get going — short articles with references and virtual machines that are all set for practice. Make sure to read everything and complete the tasks in the articles before moving on to the main challenges.
